- Javascript merupakan bahasa pemograman untuk membuat sebuah halaman web menjadi dinamis dan interaktif
- Penggunaan javascript bisa internal,eksternal
- Tipe - tipe data :
1. number
2. string
3. boolean
4. object

- variabel adalah suatu penyimpanan untuk data dan dapat digunakan untuk memanggil data tersebut
-operator aritmatika digunakan pada js untuk melakukan operasi aritmatika
- function yaitu sub-program yang bisa digunakan kembali
- event pada js adalah serangkaian perlakuan yang dapat memicu javascript
